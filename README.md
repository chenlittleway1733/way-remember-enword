# 國中背單字系統｜第一階段

## 檔案說明

- `app.py`：Streamlit 主程式
- `words.csv`：單字主檔範例
- `requirements.txt`：安裝套件

## 執行方式

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 第一階段功能

- 左側選年級 / 學期 / 單元
- 國一先修不分學期，直接選單元
- 單元可複選
- 右側顯示單字卡
- 顯示英文、詞性、中文、例句
- 瀏覽器 TTS 發音
- 上一張 / 下一張 / 隨機

## words.csv 欄位

| 欄位 | 說明 |
|---|---|
| word_id | 單字唯一編號 |
| grade | 國一先修 / 國一 / 國二 / 國三 |
| semester | 國一先修留空；其他填上學期 / 下學期 |
| unit_id | 單元代碼 |
| unit_name | 單元名稱 |
| word_order | 單字排序 |
| word | 英文單字 |
| part_of_speech | 詞性 |
| chinese | 中文意思 |
| example_1_en | 例句 1 英文 |
| example_1_zh | 例句 1 中文 |
| example_2_en | 例句 2 英文 |
| example_2_zh | 例句 2 中文 |
| example_3_en | 例句 3 英文 |
| example_3_zh | 例句 3 中文 |

## 第二階段：學習狀況 CSV

本版新增：

- 左側可上傳 `progress.csv`
- 左側可下載目前學習狀況 `progress.csv`
- 每個單字分開記錄：
  - 中翻英熟練度
  - 英翻中熟練度
  - 中翻英答對 / 答錯次數
  - 英翻中答對 / 答錯次數
  - 最後複習時間
- 若沒有上傳進度，系統會自動建立全新進度，所有單字皆為「未記憶」。
- 若上傳舊進度檔，系統會自動和目前 `words.csv` 對齊，缺少的新單字會自動補成未記憶。

熟練度數字對應：

| 數字 | 顯示 |
|---:|---|
| 0 | 未記憶 |
| 1 | 不熟 |
| 2 | 有點熟 |
| 3 | 很熟 |
| 4 | 精熟 |

## 第三階段：測驗模式

本版新增：

### 1. 測驗模式

左側模式可切換：

- 記憶模式
- 測驗模式

測驗模式可選：

- 中翻英
- 英翻中

### 2. 中翻英

- 顯示中文意思與詞性。
- 學生輸入英文。
- 可按喇叭聽該英文單字。
- 系統自動判斷是否答對。
- 答對：中翻英熟練度 +1。
- 答錯：中翻英熟練度 -1。
- 最低為未記憶，最高為精熟。

### 3. 英翻中

- 顯示英文單字。
- 可按喇叭聽發音。
- 按「公布答案」後顯示中文意思。
- 學生自行按「我答對了」或「我答錯了」。
- 答對：英翻中熟練度 +1。
- 答錯：英翻中熟練度 -1。

### 4. 精熟單字出題規則

- 預設不出精熟單字。
- 可勾選「精熟單字也列入出題」。

### 5. 學習狀況更新

測驗後會更新目前工作階段的學習狀況。
需要保存時，請按左側「下載學習狀況 CSV」。
下次使用時，再上傳該 CSV。
